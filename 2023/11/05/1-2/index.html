安装宝塔

Centos安装脚本

yum install -y wget && wget -O install.sh http://www.btkaixin.net/install/install_6.0.sh && sh install.sh

Ubuntu/Debian安装脚本

wget -O install.sh http://www.btkaixin.net/install/install_6.0.sh && bash install.sh

然后安装环境根据需要安装，推荐LNMP

然后安装tor

Centos

yum install tor

Ubuntu/Debian

apt install tor

输入y安装

打开宝塔编辑编辑tor配置文件 /etc/tor/torrc

把上面那两行前面的#字符号去掉,其他不用改

HiddenServiceDir /var/lib/tor/hidden_service/
HiddenServicePort 80 127.0.0.1:80

保存退出，重启tor服务

service tor restart

这里的80端口是可以修改的

sudo service tor start

一旦启动，那么你的域名也就生成了

查看Tor域名

cat /var/lib/tor/hidden_service/hostname

就会得出一个类似.onion 这样的域名

后缀是.onion 全部复制

宝塔新建站点

然后根目录上传网站程序，完成
